# WS2812B_Bluetooth_Controller

## 项目背景
初三下学期暑假开发的基于STC15W(8051)单片机+HC05蓝牙模块的5路WS2812B灯带控制器

## 项目组成
1. STC15W单片机程序
2. Altium Designer 10 设计的PCB
3. APP Inventor开发的安卓APP
